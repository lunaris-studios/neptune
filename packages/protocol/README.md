# Nucleus

Nucleus is a Design System built on top of Nucleus System.

### Develop

#### Install dependencies

```sh
yarn
```

#### Run Storybook

```sh
yarn storybook
```

#### Watch compiled lib

```sh
yarn watch
```

> Useful when developing in `link` mode

### Build

#### Run build

```sh
yarn build
```
